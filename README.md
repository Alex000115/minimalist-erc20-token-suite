# Minimalist ERC-20 Token Suite

This repository provides a high-quality, streamlined implementation of an ERC-20 smart contract. It is designed for developers who need a robust, audited foundation for their crypto projects without the complexity of deep folder nesting.

## Features
- **ERC-20 Compliant**: Fully compatible with the Ethereum Improvement Proposal 20.
- **OpenZeppelin Standards**: Built using industry-leading security libraries.
- **Flat Structure**: Simplified file management for browser-based development.
- **Ready for Deployment**: Includes basic configuration for Hardhat/Foundry environments.

## Quick Start
1. Copy the `MyToken.sol` contract.
2. Update the name and symbol in the constructor.
3. Deploy using the provided `deploy.js` script.
